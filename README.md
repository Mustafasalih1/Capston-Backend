# Daily Fuel & Power Reporting System 

## Project Overview
Backend system to track daily fuel distribution and power availability across multiple sites and states.

## Structure
- Django Project: `daily_fuel_report`
- Apps:
  - `users`: User management
  - `location`: States and Sites
  - `reports`: Daily fuel reports
- REST API implemented using Django REST Framework (DRF)

## Available Endpoints
- Users: `/api/users/`
- States: `/api/states/`
- Sites: `/api/sites/`
- Reports: `/api/reports/`
